## О проекте

Проект содержит автотесты для API сервиса Stellar Burgers.

Тестирование реализовано в отдельном Maven-проекте с использованием:

* JUnit 4 — для написания и запуска тестов
* REST Assured — для отправки HTTP-запросов и проверки ответов
* Allure — для формирования отчёта о выполнении тестов

### Покрытие тестами

**Создание пользователя**

* создание уникального пользователя
* создание уже зарегистрированного пользователя
* создание пользователя без обязательного поля

**Логин пользователя**

* вход под существующим пользователем
* вход с неверными учетными данными

**Создание заказа**

* создание заказа с авторизацией
* создание заказа без авторизации
* создание заказа с ингредиентами
* создание заказа без ингредиентов
* создание заказа с некорректным хешем ингредиентов
